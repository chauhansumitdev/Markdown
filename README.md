# Markdown All in One
## Headers
```
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6
```

## Emphasis

```
*italic* or _italic_
**bold** or __bold__

```

## Lists
### Unordered List
```
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

```
### Ordered List
```
1. Item 1
2. Item 2
   1. Subitem 2.1
   2. Subitem 2.2

```
## Link/Image
```
[Link Text](http://www.example.com)
![Alt Text](image.jpg)
```
## BlockQuote
> Example - This is a blockquote.
```
> This is a blockquote.

```

## Code
```
`inline code`

\```python
def example_function():
    print("Hello, Markdown!")
``` <- Block Code

```

## Horizontal Line/Rule

```
---
```

## Tables

```
| Header 1 | Header 2 |
| ---------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

```

## Task List
 Example ->
 - [x] Task 1
- [ ] Task 2
- [ ] Task 3

```
- [x] Task 1
